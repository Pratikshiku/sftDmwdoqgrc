### 一、相关技术
- 后端：`Java`、`JavaWeb` / `Springboot`。
- 前端：`Vue`、`HTML` / `CSS` / `Javascript` 等。
- 数据库：`MySQL`

- `Eclipse`
- `Visual Studio Code(VScode)`

### 三、功能描述
系统分为（身份）：普通用户(学生）、教师、管理员。

**普通用户功能：**
1. 登录
2. 注册
3. 修改密码
4. 个人信息
5. 通知公告
6. 课程信息
7. 学生课程


**教师功能：**
1. 登录
2. 注册
3. 修改密码
4. 个人信息
5. 通知公告
6. 学生信息
7. 课程信息
8. 学生课程



**管理员功能：**
1. 登录
2. 个人中心
3. 通知公告管理
4. 学生管理
5. 教师管理
6. 课程信息管理
7. 学生课程管理
